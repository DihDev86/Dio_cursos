Posicionamento das imagens de fundo

Para quê serve o background-position?

O background-position é uma propriedade CSS que define a posição inicial de uma imagem de fundo. A imagem de fundo é colocada no contêiner de acordo com os valores fornecidos para o background-position.

Quais são os valores e como são usados?

Os valores do background-position podem ser expressos em pixels, porcentagem ou palavras-chave. Os valores são usados para definir a posição da imagem de fundo em relação ao contêiner.

Palavras-chave:

  • top: a imagem de fundo é posicionada no topo do contêiner.
  • bottom: a imagem de fundo é posicionada na parte inferior do contêiner.
  • left: a imagem de fundo é posicionada à esquerda do contêiner.
  • right: a imagem de fundo é posicionada à direita do contêiner.
  • center: a imagem de fundo é posicionada no centro do contêiner.

Usando dois valores:

Se você fornecer dois valores para o background-position, o primeiro valor será a posição horizontal e o segundo valor será a posição vertical da imagem de fundo.

Exemplo: background-position: 50% 50%; - a imagem de fundo é centralizada no contêiner.

Usando valores em pixels e porcentagem

Você também pode usar valores em pixels e porcentagem para definir a posição da imagem de fundo.

Exemplo: background-position: 100px 50%; - a imagem de fundo é posicionada 100 pixels à direita e centralizada verticalmente no contêiner.

Nota: Se você fornecer apenas um valor, ele será considerado como a posição horizontal da imagem de fundo. A posição vertical será definida como center por padrão.
