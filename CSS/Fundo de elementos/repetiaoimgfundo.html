Repetição das imagens de fundo

A propriedade background-repeat controla como a imagem de fundo se repete no contêiner (horizontalmente, verticalmente ou nenhum dos dois). No entanto, se a imagem for muito grande ou muito pequena em relação ao contêiner, pode parecer que background-repeat não está funcionando.

propriedade background-size! Ela ajusta o tamanho da imagem de fundo para que fique proporcional ou no tamanho correto dentro do contêiner. Isso garante que o comportamento do background-repeat seja visível e previsível.

background-repeat: repeat (padrão)

A imagem de fundo é repetida horizontalmente e verticalmente para cobrir todo o contêiner.

background-repeat: repeat-x

A imagem de fundo é repetida apenas horizontalmente.

background-repeat: repeat-y

A imagem de fundo é repetida apenas verticalmente.

background-repeat: no-repeat

A imagem de fundo não é repetida. Apenas uma instância da imagem será exibida.

background-repeat: space

  • A imagem de fundo é repetida sem cortar nenhuma parte.
  • Espaços uniformes são adicionados entre as repetições para preencher o contêiner.
  • A imagem é ajustada para que haja um espaço igual entre as cópias.

background-repeat: round

  • A imagem de fundo é repetida e redimensionada para se ajustar ao contêiner.
  • Nenhum espaço vazio será deixado, e a imagem é redimensionada para caber perfeitamente nas dimensões disponíveis.
