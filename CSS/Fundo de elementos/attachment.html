Propriedade background Attachment

Para quê serve o background-attachment?

O background-attachment é uma propriedade CSS que define se a imagem de fundo rola com o conteúdo da página ou se ela permanece fixa.

Quais são os valores e como são usados?

Os valores do background-attachment podem ser scroll ou fixed. O valor scroll faz com que a imagem de fundo role com o conteúdo da página, enquanto o valor fixed faz com que a imagem de fundo permaneça fixa.

Exemplo:

.exemplo {
background-image: url('./img/montanha.jpg');
background-attachment: fixed;
}

background-attachment: fixed

Neste exemplo, a imagem de fundo permanece fixa enquanto o conteúdo da página rola.

background-attachment: fixed;
