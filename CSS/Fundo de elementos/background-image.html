background-image

A propriedade background-image no CSS é usada para definir uma ou mais imagens de fundo em um elemento. Essas imagens são renderizadas no plano de fundo do elemento, abaixo do conteúdo e das bordas.

A sintaxe básica é:

background-image: url('caminho/para/imagem.jpg');

Você pode usar diferentes valores para personalizar o plano de fundo.

Valores possiveis de background-image

  1. Url( )

    Define uma imagem como plano de fundo usando o caminho do arquivo. Pode ser um caminho relativo ou absoluto:

    Valor absoluto é o valor que vem de uma url de um arquivo da máquina.

    background-image: url('./imagens/fundo.jpg');

    Valor relativo é o valor que vem de uma url externa, ou seja, de algum link da internet.

    background-image: url('https://exemplo.com/imagem.jpg');

  2. none

    Remove a imagem de fundo do elemento. O plano de fundo será transparente.

    background-image: none;

  3. linear-gradient( )

    Cria uma imagem de fundo com um gradiente linear. Você pode especificar duas ou mais cores e suas posições.

    background-image: linear-gradient(lightpink, lightgreen);

  4. radial-gradient( )

    Cria uma imagem de fundo com um gradiente radial. Você pode especificar duas ou mais cores e suas posições.

    background-image: radial-gradient(lightgreen, lightskyblue);

  5. repeating-linear-gradient

    Cria uma imagem de fundo com um gradiente linear que se repete. Você pode especificar duas ou mais cores e suas posições.

    background-image: repeating-linear-gradient(to top,palevioletred 0 20px , lightpink 20px 40px );
