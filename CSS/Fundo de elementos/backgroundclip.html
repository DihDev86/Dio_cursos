Propriedade background-clip

A propriedade background-clip CSS especifica se uma imagem de fundo é cortada para caber nas bordas da caixa de conteúdo, das bordas da caixa de preenchimento ou das bordas da caixa de borda.

A propriedade background-clip tem três valores possíveis:border-box, padding-box e content-box.

A propriedade background-clip tem algumas semelhanças com a background-origin. A diferença é que a background-origin define a origem da imagem de fundo, enquanto a background-clip define a área de exibição da imagem de fundo.

background-clip: padding-box;

Se você definir a propriedade background-clip como padding-box, a imagem de fundo será cortada para caber nas bordas da caixa de preenchimento.

background-clip: border-box;

Se você definir a propriedade background-clip como border-box, a imagem de fundo será cortada para caber nas bordas da caixa de bordas.

background-clip: content-box;

Se você definir a propriedade background-clip como content-box, a imagem de fundo será cortada para caber nas bordas da caixa de conteúdo.
