Propriedade object-fit

A propriedade object-fit no CSS é usada para especificar como o conteúdo de um elemento, como uma imagem ou vídeo, deve se ajustar à caixa do seu contêiner.

Sintaxe

CSS

elemento {
object-fit: valor;
}


Objetc-fit: ;

Valores possíveis do object-fit

  1. fill (padrão):

    Faz com que o conteúdo preencha completamente o contêiner, mesmo que precise distorcer as proporções.

    img {
    object-fit: fill;
    }

  2. contain:

    Redimensiona o conteúdo para caber completamente no contêiner, mantendo suas proporções originais.

    Pode deixar espaços vazios ao redor do conteúdo.

    img {
    object-fit: contain;
    }

  3. cover:

    Redimensiona o conteúdo para preencher completamente o contêiner, mantendo as proporções originais.

    Parte do conteúdo pode ser cortada se não couber.

    img {
    object-fit: cover;
    }

  4. none:

    Mantém o tamanho original do conteúdo sem redimensioná-lo.

    Parte do conteúdo pode ficar cortada ou transbordar do contêiner.

    img {
    object-fit: none;
    }

  5. scale-down:

    Funciona como uma combinação de none e contain.

    O conteúdo é exibido no menor tamanho possível entre esses dois valores.

    img {
    object-fit: scale-down;
    }

Quando usar object-fit

  • Para lidar com imagens responsivas.
  • Ajustar vídeos em diferentes tamanhos de contêineres.
  • Evitar distorções em imagens ao usar layouts flexíveis.
