Propriedade objetc-position

A propriedade object-position no CSS é usada para definir o alinhamento do conteúdo de um elemento (como uma imagem ou vídeo) dentro do contêiner, quando ele não preenche o contêiner totalmente. Ela funciona em conjunto com object-fit.

Sintaxe

elemento {
object-position: valor-x valor-y;
}

Por padrão, o valor é 50% 50%, que posiciona o conteúdo no centro.

  • valor-x: Alinha o conteúdo horizontalmente (esquerda, centro, direita).
  • valor-y: Alinha o conteúdo verticalmente (topo, centro, base).

Valores Aceitos

1.Palavra-chave:

  • top, bottom → Posicionam verticalmente no topo ou na base.
  • left, right → Posicionam horizontalmente à esquerda ou à direita.
  • center → Centraliza o conteúdo horizontal ou verticalmente.

Exemplo:

img {
object-position: left top; /* Alinhado ao canto superior esquerdo */
}

2.Porcentagens:

  • Especifica a posição como uma porcentagem da largura e altura do contêiner.
  • 0% → Início do eixo.
  • 100% → Final do eixo.

Exemplo:

img {
object-position: 25% 75%; /* 25% do eixo X, 75% do eixo Y */
}

3.Valores em pixels:

  • Pode usar valores fixos, como px, para posicionar o conteúdo de forma absoluta dentro do contêiner.

Exemplo:

img {
object-position: 10px 20px; /* 10px da esquerda, 20px do topo */
}

Quando usar object-position?

  • Para controlar qual parte do conteúdo será visível em contêineres com cortes (como imagens cover ou contain).
  • Para alinhar elementos em layouts esteticamente agradáveis.
