Agrupamento de Seletores

Podemos aplicar as mesmas regras CSS para vários seletores diferentes, para isso podemos separar os seletores usando virgula(,)

Assim, não precisamos ficar copiando e colando os mesmos códigos para todos, nós podemos fazer um só bloco de CSS e reutilizar as regra.
