Seletores CSS

Bem-vindo!

No CSS nós possuimos seletores:

  • Seletor por tipo: Busca elementos por uma tag HTML espicífica
  • Seletor por ID #: Seleciona um elemento com um ID específico (único na página).
  • Seletor universal (*): Seleciona todos os elementos da página.
  • Seletores de atributos([atributo]) Seleciona elementos que possuem um atributo específico.
  • Seletores de classe (.): Seleciona elementos que possuem uma classe específica.
