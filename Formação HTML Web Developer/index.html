Desafio com Tags HTML(DIO)

Tag <P>

A tag p em HTML é usada para definir parágrafos de texto. Ela é uma das tags mais básicas e essenciais para a marcação de conteúdo textual em uma página web. Esse texto foi feito com o uso da tag <P> .

Ela deve sempre ser usada com uma tag de abertura <p> e uma tag de fechamento </p>, onde o conteúdo será escrito dentro das duas tags.

Tags de títulos

As tags de título em HTML são usadas para definir títulos e subtítulos em um documento web. Elas vão de h1 a h6, onde h1 representa o título mais importante e <h6> o menos importante. Veja o exemplo abaixo seguindo a ordem, ou seja, da <h1> até a <h6>:

<h1>Título</h1>

<h2>Título</h2>

<h3>Título</h3>

<h4>Título</h4>

<h5>Título</h5>
<h6>Título</h6>

Tag <mark>

A tag <mark> em HTML é usada para destacar partes específicas de um texto, geralmente com um fundo amarelo, simulando um marcador de texto físico. Ela é útil para enfatizar informações importantes ou realçar palavras-chave.

Para destacar partes específicas de um texto em HTML, deve-se utilizar a tag <mark> com abertura <mark>e fechamento </mark>. Essa tag é útil para realçar informações importantes dentro de um conteúdo, tornando-as visualmente distintas. É importante usar a tag <mark> de forma semântica, reservando-a para destacar palavras-chave ou trechos significativos no contexto do texto. Abaixo temos um texto usando a tag <mark>,

ipsum dolor ipsum dolor sit amet consectetur adipisicing elit. Pariatur quod quia harum et quas ratione voluptate? Accusamus, ad autem aspernatur iusto molestias facere in quasi delectus voluptatibus harum optio deleniti.

Tag <small>

A tag <small> em HTML é usada para indicar que o texto contido dentro dela deve ser renderizado em um tamanho de fonte menor do que o texto circundante. A tag com tag de abertura <small> e fechamento </small>. Abaixo temos um exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Quos sunt reiciendis perferendis eos molestiae, nesciunt fugiat explicabo perspiciatis quo, quaerat aut eum exercitationem nam unde ad iste quibusdam, asperiores ea!

Tag <i>

A tag <i> em HTML é utilizada para indicar que o texto dentro dela deve ser renderizado em itálico. Anteriormente, ela era frequentemente usada para destacar palavras ou frases em itálico, especialmente quando o significado ou ênfase era diferente do texto circundante. No entanto, é importante notar que, segundo as diretrizes atuais de HTML5, a tag <i> é considerada obsoleta para esse propósito semântico. Em vez disso, recomenda-se usar CSS para aplicar estilos de texto, como itálico, e reservar as tags HTML para estruturar o conteúdo de forma mais semântica.

A tag <i> é usada dessa forma <i>conteúdo</i>. Abaixo temos um texto com o uso da tag:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Rerum culpa perspiciatis, quisquam praesentium pariatur fuga non odio quaerat cum dignissimos dicta. Esse veniam quam maxime, illum dolor earum laborum beatae!

Tag <u>

A tag <u> em HTML é usada para sublinhar texto. Ela foi amplamente utilizada no passado para esse fim, mas, assim como a tag <i> para itálico, a tag <u> também é considerada obsoleta no HTML5 em termos de semântica. Recomenda-se, atualmente, usar CSS para estilizar o texto, incluindo a aplicação de sublinhado, e reservar as tags HTML para estruturar o conteúdo de forma mais semântica.

A tag <u> é usada dessa forma <u>conteúdo</u>. Abaixo temos um exemplo seu uso:

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Debitis labore eum molestias, vero aperiam quia repudiandae deserunt, est ex odit magni voluptas cumque culpa itaque dolores quae minima id doloremque.

Tag <strong>

A tag <strong> em HTML é usada para indicar que o texto dentro dela deve ser considerado semanticamente forte ou enfatizado. Anteriormente, era comumente interpretada pelo navegador para exibir o texto em negrito, mas seu propósito real é conferir ênfase significativa ao conteúdo, não apenas estilístico. A tag <strong> é usada dessa forma <strong>conteúdo</strong>, abaixo temos um exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Voluptas, ipsa saepe culpa id tempore sed quis nostrum! Officia, repudiandae. Est explicabo magni voluptas eum distinctio! Officia saepe alias dicta hic.
