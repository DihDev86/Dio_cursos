Desafio com Tags HTML(DIO)

Tag <P>

A tag p em HTML é usada para definir parágrafos de texto. Ela é uma das tags mais básicas e essenciais para a marcação de conteúdo textual em uma página web. Esse texto foi feito com o uso da tag <P> .

Ela deve sempre ser usada com uma tag de abertura <p> e uma tag de fechamento </p>, onde o conteúdo será escrito dentro das duas tags.

Tags de títulos

As tags de título em HTML são usadas para definir títulos e subtítulos em um documento web. Elas vão de h1 a h6, onde h1 representa o título mais importante e <h6> o menos importante. Veja o exemplo abaixo seguindo a ordem, ou seja, da <h1> até a <h6>:

<h1>Título</h1>

<h2>Título</h2>

<h3>Título</h3>

<h4>Título</h4>

<h5>Título</h5>
<h6>Título</h6>

Tag <mark>

A tag <mark> em HTML é usada para destacar partes específicas de um texto, geralmente com um fundo amarelo, simulando um marcador de texto físico. Ela é útil para enfatizar informações importantes ou realçar palavras-chave.

Para destacar partes específicas de um texto em HTML, deve-se utilizar a tag <mark> com abertura <mark>e fechamento </mark>. Essa tag é útil para realçar informações importantes dentro de um conteúdo, tornando-as visualmente distintas. É importante usar a tag <mark> de forma semântica, reservando-a para destacar palavras-chave ou trechos significativos no contexto do texto. Abaixo temos um texto usando a tag <mark>,

ipsum dolor ipsum dolor sit amet consectetur adipisicing elit. Pariatur quod quia harum et quas ratione voluptate? Accusamus, ad autem aspernatur iusto molestias facere in quasi delectus voluptatibus harum optio deleniti.

Tag <small>

A tag <small> em HTML é usada para indicar que o texto contido dentro dela deve ser renderizado em um tamanho de fonte menor do que o texto circundante. A tag com tag de abertura <small> e fechamento </small>. Abaixo temos um exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Quos sunt reiciendis perferendis eos molestiae, nesciunt fugiat explicabo perspiciatis quo, quaerat aut eum exercitationem nam unde ad iste quibusdam, asperiores ea!

Tag <i>

A tag <i> em HTML é utilizada para indicar que o texto dentro dela deve ser renderizado em itálico. Anteriormente, ela era frequentemente usada para destacar palavras ou frases em itálico, especialmente quando o significado ou ênfase era diferente do texto circundante. No entanto, é importante notar que, segundo as diretrizes atuais de HTML5, a tag <i> é considerada obsoleta para esse propósito semântico. Em vez disso, recomenda-se usar CSS para aplicar estilos de texto, como itálico, e reservar as tags HTML para estruturar o conteúdo de forma mais semântica.

A tag <i> é usada dessa forma <i>conteúdo</i>. Abaixo temos um texto com o uso da tag:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Rerum culpa perspiciatis, quisquam praesentium pariatur fuga non odio quaerat cum dignissimos dicta. Esse veniam quam maxime, illum dolor earum laborum beatae!

Tag <u>

A tag <u> em HTML é usada para sublinhar texto. Ela foi amplamente utilizada no passado para esse fim, mas, assim como a tag <i> para itálico, a tag <u> também é considerada obsoleta no HTML5 em termos de semântica. Recomenda-se, atualmente, usar CSS para estilizar o texto, incluindo a aplicação de sublinhado, e reservar as tags HTML para estruturar o conteúdo de forma mais semântica.

A tag <u> é usada dessa forma <u>conteúdo</u>. Abaixo temos um exemplo seu uso:

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Debitis labore eum molestias, vero aperiam quia repudiandae deserunt, est ex odit magni voluptas cumque culpa itaque dolores quae minima id doloremque.

Tag <strong>

A tag <strong> em HTML é usada para indicar que o texto dentro dela deve ser considerado semanticamente forte ou enfatizado. Anteriormente, era comumente interpretada pelo navegador para exibir o texto em negrito, mas seu propósito real é conferir ênfase significativa ao conteúdo, não apenas estilístico. A tag <strong> é usada dessa forma <strong>conteúdo</strong>, abaixo temos um exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Voluptas, ipsa saepe culpa id tempore sed quis nostrum! Officia, repudiandae. Est explicabo magni voluptas eum distinctio! Officia saepe alias dicta hic.

Tag <ol>

A tag <ol> em HTML é utilizada para criar listas ordenadas, ou seja, listas cujos itens são numerados ou ordenados de alguma forma sequencial.

Listas Numeradas: tag <ol> envolve uma lista de itens que serão exibidos com números, letras ou outro tipo de marcador sequencial.
Ordem Semântica: É usada quando a ordem dos itens é importante, como em instruções passo a passo, rankings, procedimentos, entre outros.

A sintaxe básica da tag <ol> é a seguinte:

<ol>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ol>

<li> : Tags de item de lista que contêm os itens individuais da lista. Cada <li> deve estar dentro da tag <ol> .

  1. Item 1
  2. Item 2
  3. Item 3

Tag <ul>

A tag <ul> em HTML é utilizada para criar listas não ordenadas, ou seja, listas cujos itens não possuem uma ordem específica e são geralmente marcados com pontos (bullets).

<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>

<li> : Tags de item de lista que contêm os itens individuais da lista. Cada <li> deve estar dentro da tag <ul> .

  • Item 1
  • Item 2
  • Item 3

Tag <a>

A tag <a> em HTML é utilizada para criar hiperlinks, que são elementos clicáveis que direcionam o usuário para outra página da web, um recurso específico, uma âncora dentro da mesma página, ou qualquer outro destino, como arquivos ou endereços de e-mail.

Utilização

Hiperlinks: A tag <a> define um link de ancoragem. O destino do link é especificado pelo atributo href.
Navegação: É essencial para a navegação entre páginas e recursos na web.

Sintaxe

<a href="URL">Texto do Link/a>

tag <a>:Tag de abertura que define o início de um hiperlink.
href: Atributo que especifica o destino do link. Pode ser uma URL, uma âncora interna, um endereço de e-mail, etc.
Texto do Link: O texto ou conteúdo (pode ser texto, imagem, etc.) que será clicável.
</a>: Tag de fechamento que define o final do hiperlink.

Tag <hr>

A tag <hr> em HTML é usada para inserir uma linha horizontal na página. Essa linha é geralmente usada para separar conteúdo ou seções de um documento de forma visual. A tag <hr> é uma tag auto-fechada, o que significa que ela não precisa de uma tag de fechamento.

Utilização

Separação de Conteúdo: A tag <hr> é comumente usada para criar uma separação visual entre diferentes seções ou temas de conteúdo dentro de uma página.
Elementos Horizontais: Ela pode ser estilizada usando CSS para ajustar sua aparência de acordo com as necessidades do design da página.

Tag <sub>

A tag <sub>em HTML é usada para definir texto que deve ser exibido como subscript (subscrito), ou seja, texto que é exibido abaixo da linha de base do texto circundante. Essa formatação é comum em notações científicas e matemáticas, onde é necessário indicar índices, como em fórmulas químicas ou expressões matemáticas.

Utilização

Texto Subscrito:A tag <sub> envolve o texto que você deseja exibir como subscript.
Notações Científicas e Matemáticas: É frequentemente usada para indicar índices em fórmulas químicas, notações matemáticas, e outras situações onde o texto deve aparecer abaixo da linha de base do texto normal.

Sintaxe

A sintaxe básica da tag <sub> é a seguinte:

<sub>Texto Subscrito</sub>

exemplo

A fórmula da água é H2O.

Tag <sup>

A tag <sup> em HTML é usada para definir texto que deve ser exibido como superscript (sobrescrito), ou seja, texto que é exibido acima da linha de base do texto circundante. Essa formatação é comum em notações científicas e matemáticas, onde é necessário indicar expoentes, como em fórmulas químicas, expressões matemáticas e referências bibliográficas.

Sintaxe

A sintaxe básica da tag <sup> é a seguinte:

<sup>Texto Sobrescrito</sup>

Exemplo

A fórmula da área do quadrado é A = s2.

Tag <blockquote>

A tag <blockquote> em HTML é usada para definir uma seção que é citada de outra fonte. Ela é utilizada para delimitar citações longas que geralmente são apresentadas como um bloco separado do texto principal. O navegador renderiza essas citações com recuo (indentação), destacando-as do restante do conteúdo.

Utilização

Citações Longas: A tag <blockquote> é ideal para incluir citações extensas no conteúdo.
Recuo Automático: Navegadores aplicam um estilo de recuo automático para distinguir visualmente a citação do restante do texto.

Sintaxe

A sintaxe básica da tag <blockquote> é a seguinte:

<blockquote>Texto da citação.</blockquote>

Exemplo
